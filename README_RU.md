# Second Brain OS — личное приложение

Это PWA-приложение: один код работает на ПК и iPhone через браузер. Его можно установить на главный экран iPhone как приложение и на рабочий стол ПК через Chrome/Edge.

## Что внутри

- Главная панель Life Score
- Финансы, доходы, расходы, распределение 10/10/10/70
- CSV-импорт банковских выписок с ручным назначением категорий
- Панель анализа за любой период
- SMART-цели и связь с задачами
- Управление привычками: добавление и скрытие
- Задачи, дедлайны, просрочки
- Состояние: сон, энергия, настроение, стресс
- Закрытие дня
- Недельные отчеты
- Инсайты, достижения, антихаос-режим
- Бэкап и восстановление данных JSON

## Как быстро открыть на ПК

1. Распакуйте архив.
2. Откройте файл `index.html` двойным кликом.
3. Для нормального PWA-режима лучше запустить локальный сервер:

```bash
python -m http.server 5173
```

Потом открыть:

```text
http://localhost:5173
```

## Как поставить на iPhone без App Store

Самый простой путь — выложить папку на GitHub Pages или Netlify, потом открыть ссылку в Safari и добавить на экран «Домой».

### Через GitHub Pages

1. Создайте аккаунт GitHub.
2. Создайте новый репозиторий, например `second-brain-os`.
3. Загрузите все файлы из папки приложения.
4. Откройте Settings → Pages.
5. В Source выберите `Deploy from branch`, Branch `main`, folder `/root`.
6. Дождитесь ссылки вида `https://ваш-логин.github.io/second-brain-os/`.
7. Откройте эту ссылку на iPhone в Safari.
8. Нажмите «Поделиться» → «Добавить на экран Домой».

## Где хранятся данные

Данные хранятся локально в браузере в `localStorage`. Это удобно для личного использования, но важно делать бэкап:

- Настройки → Скачать бэкап
- Настройки → Загрузить бэкап

## Что делать, если данные пропали

1. Проверьте, что открываете приложение по той же ссылке/адресу.
2. Восстановите JSON-бэкап через Настройки.

## Как сделать настоящую нативную iOS-версию позже

Эту PWA можно обернуть в Capacitor. Тогда приложение откроется в Xcode и его можно будет установить на iPhone как нативное приложение. Для публикации в App Store нужен Apple Developer Program, но для личного теста обычно хватает запуска через Xcode на свое устройство.
