# Second Brain OS V42 — Product Engine

Версия V42 добавляет продуктовый слой поверх V41 без переписывания старого ядра.

## Что добавлено

- Manager Dashboard: фокус дня, клиенты, импорт, качество данных, быстрые действия.
- Category Center: единый раздел категорий для финансов, задач, клиентов, целей и импорта.
- Автоправила категорий для банковского импорта.
- CRM Light: клиенты, статусы, сумма сделки, следующий контакт, связанные задачи.
- Быстрый ввод V42: текстовая команда превращается в задачу, расход/доход или клиента.
- Диагностика V42: проверка кнопок, данных, категорий, localStorage, импортов и service worker.
- Журнал действий и отмена последнего импорта.
- Мобильная навигация с быстрым доступом к клиентам.
- Новый cache-key service worker для сброса старой V41/V40 логики.
