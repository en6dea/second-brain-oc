# Second Brain OS V86.9 — исправление финансов и планирование по месяцам

Основа сборки: настоящая V86.7 Finance Control.

Сохранено из V86.7:
- лимиты категорий;
- обязательные и необязательные расходы;
- финансовый помощник;
- подробная аналитика;
- проверка банковского CSV;
- редактирование категорий.

Добавлено:
- безопасное открытие всех финансовых маршрутов;
- папка «Планирование»;
- план расходов по любому месяцу;
- повторение планового расхода на будущие месяцы;
- обязательность, категория, сумма, день оплаты и комментарий;
- прогноз остатка;
- анализ будущей нагрузки на лимиты категорий;
- проведение планового расхода в реальные операции;
- копирование расхода на следующий месяц;
- новая премиальная плашка прогресса привычек.

Данные не очищаются. Новые записи хранятся в financePlans.
