# Second Brain OS V32 — Root Cause Fix кнопок

Исправлена причина, из-за которой кнопки перестали работать после V31.

Проблема: общий render ссылался на `polinaPage`, но эта функция была объявлена внутри отдельного модуля и не была доступна новому календарному/кнопочному роутеру. Из-за ошибки `polinaPage is not defined` любой клик, который вызывал render, падал и создавал ощущение, что не работает всё приложение.

Что изменено:
- добавлен уникальный файл `app-v32-root-cause-fix.js`;
- `index.html` подключает именно его;
- все карты render используют безопасный fallback для страницы Полины;
- дизайн, данные и логика разделов не менялись.
