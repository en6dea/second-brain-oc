# Second Brain OS V49 — Focus Path Goals

Чистая интеграция в последнюю рабочую сборку без удаления текущих разделов и данных.

## Что добавлено

1. Новая страница **Фокус дня** — пошаговый маршрут: утренний фокус → цель → задачи → привычки → финансы → вечернее закрытие.
2. Новая страница **Входящие** — быстрый захват мыслей, задач, заметок, покупок, долгов, желаний и людей.
3. Новая страница **Обзоры** — ежедневное закрытие, недельная проверка и колесо жизни.
4. Новая лента задач с режимами: Сегодня, Эта неделя, По целям, Быстрые, Ожидаю, Когда-нибудь, Все.
5. Улучшенные карточки задач: выполнить, разбить, сделать шагом недели, перенести на завтра, ожидание, Google, редактирование.
6. Улучшенные SMART-цели: подцели, шаги, прогресс результата, прогресс действий, вопрос недели и мастер разложения цели.
7. Антипрокрастинационный блок: быстрый шаг на 15 минут и мягкая последовательность действий.
8. Версточные фиксы: более устойчивые сетки, аккуратные карточки, меньше расползания списков.

## Важно

Старые разделы не удалены: финансы, долги, задачи, планирование, покупки, желания, личное, люди, заметки, идеи, привычки, цели, документы, книги, фильмы, путешествия, архив, диагностика и другие модули остаются.

## Проверка после загрузки

Открыть:

```text
https://en6dea.github.io/second-brain-oc/?v=second-brain-space-v49-focus-path-goals-20260708#focus-path
```

Проверить:

- в левом меню появились **Фокус дня**, **Входящие**, **Обзоры**;
- открывается маршрут Фокус дня;
- задачи переключаются по вкладкам;
- цель можно разбить через **Мастер цели**;
- у цели создаются подцели и шаг недели;
- входящие сохраняются и переносятся в систему;
- старые финансы, долги и CSV-категории остались рабочими;
- внизу слева бейдж: `V49 · FOCUS PATH GOALS`.

## Если отображается старая версия

Открыть диагностику:

```text
diagnostics-v49.html
```

или выполнить в консоли браузера:

```js
navigator.serviceWorker.getRegistrations().then(r => r.forEach(x => x.unregister()));
caches.keys().then(keys => keys.forEach(k => caches.delete(k)));
location.reload(true);
```
