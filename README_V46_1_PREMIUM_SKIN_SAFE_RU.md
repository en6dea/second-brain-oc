Безопасная визуальная сборка поверх стабильной V46.

## Главное

Эта версия НЕ меняет бизнес-логику приложения.

Изменено только:
- визуальный слой внутри `index.html`;
- cache name в `sw.js`, чтобы браузер подтянул свежий интерфейс;
- `app.js` приложен без изменений из стабильной V46 для страховки.

- импорт;
- localStorage;
- новые сущности `lifeEntries` не добавлялись.
