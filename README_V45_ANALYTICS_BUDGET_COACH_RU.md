## Новые сценарии

### Бюджет

1. Открыть раздел **Бюджет**.
2. Нажать **Автолимиты** — приложение создаст лимиты по категориям, где уже есть траты.
3. Изменить лимиты вручную.
4. Нажать **Проверить бюджет**.

### Аналитика

1. Открыть раздел **Аналитика**.
2. Посмотреть факт месяца, прогноз и структуру расходов.
3. Если прогноз выше дохода — снизить топ-категорию на 10% на этой неделе.

### Финансовая грамотность

V45 не даёт инвестиционных рекомендаций. Она помогает с бытовым управлением деньгами: вести факт, задавать лимиты, планировать, видеть перерасход и тренировать привычку контроля.
