# Установка V86 в GitHub 1. В текущем приложении скачайте JSON через «Настройки → Резервная копия». 2. Распакуйте архив update-only. 3. Загрузите его содержимое в корень репозитория с заменой файлов. 4. Папки `.github` и `scripts` также должны попасть в репозиторий. 5. Сделайте Commit changes. 6. Дождитесь завершения GitHub Actions `Second Brain OS QA`. 7. После публикации один раз откройте `force-update.html`. ## Проверка - внизу приложения должна отображаться версия `V86 · AUTOPILOT`; - на каждом экране должна быть плавающая кнопка «Зафиксировать»; - в настройках должен появиться блок «Автопилот дня»; - старые данные не должны исчезнуть.
