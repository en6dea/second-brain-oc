# Second Brain OS V48 — Clean Baseline + Rollback Safe

Эта сборка очищает проект от накопления старых видимых версий и синхронизирует `index.html` с актуальным движком.

## Что исправлено

- `index.html` теперь содержит тот же движок, что и `app.js`;
- старая плашка V41–V46 больше не должна мигать и оставаться финальной;
- обновлены `manifest.webmanifest`, `sw.js`, `offline.html`;
- добавлен V48 Version Guardian;
- сохранены функции V47: Жизнь, lifeEntries, универсальные записи, финансы, бюджет, дорожная карта.

## Как ставить безопасно

1. Сначала скачайте текущий репозиторий как ZIP или создайте ветку `backup-before-v48-clean`.
2. Затем можно удалить старые служебные README и лишние файлы из списка `DELETE_OLD_FILES_V48.txt`.
3. Загрузите файлы из clean-пакета.
4. Откройте приложение с параметром `?v=v48-clean-baseline-private-20260630&hard=1`.

## Откат

Откат возможен через GitHub: вкладка Commits → выбрать предыдущий коммит → Revert. Также можно загрузить обратно ZIP резервной копии.
