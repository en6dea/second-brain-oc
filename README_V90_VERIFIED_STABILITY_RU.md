# Second Brain OS V90 — Verified Stability

**Сборка:** `second-brain-space-v90-verified-stability-20260723-r2`  
**Версия приложения:** `90.0.2`  
**Дата:** 23.07.2026

V90 — безопасная итерация поверх существующего Second Brain OS. Приложение не переписано с нуля, ключ основного хранилища не изменён, пользовательские данные не заменяются демоданными.

## Что исправлено

### Навигация и иконки

- Зафиксирована ширина бокового меню и размеры контейнеров иконок.
- Устранены скачки размеров при переходах между разделами.
- Активный пункт получает контрастную белую иконку.
- Основная навигация, «Информация», привычки и очередь используют единый локальный SVG-набор без асинхронной подгрузки.
- Текущая сборка показывается меткой `V90 · VERIFIED STABILITY`.

### Очередь «Разобрать»

- Внутренний технический тип `transfer` больше не выводится пользователю.
- Карточки получили устойчивую grid-компоновку и автоматическую высоту.
- Текст и блок действий не пересекаются.
- Кнопка «Скрыть» обновляет очередь без перезагрузки.

### Календарь

- Отключено разбиение названий по одной букве.
- Недельная сетка получила минимальную ширину и внутреннюю горизонтальную прокрутку на узком экране.
- Длинное название отображается максимум в 3 строки.
- Событие содержит только 2 компактных действия, не зажимающих текст.
- Работают перенос между днями, изменение названия и продолжительности.
- Удаление создаёт архивную копию и показывает Undo.
- Добавлено обратносуместимое поле `durationMinutes` без замены существующих задач.

### Финансы и модальные окна

- Поля модальных форм принудительно возвращены в рабочее интерактивное состояние.
- Устранены проблемы с `pointer-events`, фокусом и перекрывающим слоем.
- Работают ввод текста и чисел, сохранение, повторное открытие значений и закрытие по Escape.
- Модальное окно фиксировано поверх страницы, блокирует прокрутку фона и удерживает фокус внутри.

### Привычки

- Выбор иконки обновляет скрытое значение и SVG-предпросмотр.
- Выбранная иконка сохраняется в модели привычки и отображается на карточке после повторного рендера.
- Старые emoji-значения остаются совместимыми и преобразуются в единый SVG-стиль на уровне интерфейса.

### PWA и кэш

- Service Worker переведён на кэш `second-brain-os-v90-verified-stability-r2`.
- При активации удаляются устаревшие кэши интерфейса.
- Регистрация Service Worker использует `updateViaCache: 'none'`.
- Навигационные запросы не должны возвращать старый HTML из предыдущей версии.

## Сохранность данных

- Основной ключ сохранён: `secondBrainOS.v1`.
- Нет автоматической очистки `localStorage` или удаления пользовательской IndexedDB.
- При первом запуске V90 выполняется попытка создать:
  - исходную копию JSON в `secondBrainOS.v90.rawLocalBackup`;
  - снимок состояния в IndexedDB `SecondBrainOSDurableStorage`, store `records`;
  - метку выполнения `secondBrainOS.v90.backupCreated`.
- Изменение схемы ограничено необязательным полем продолжительности календарной задачи.

Среда сборки блокировала прямую навигацию браузера на локальный HTTP-сервер, поэтому интерфейс тестировался в Chromium с локально встроенными файлами CSS/JS. Реальный жизненный цикл Service Worker на GitHub Pages и резервная копия фактической базы пользователя должны быть дополнительно проверены уже после публикации. Код регистрации и кэширования проверен статически.
