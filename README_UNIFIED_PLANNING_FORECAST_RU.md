# Second Brain OS — Unified Planning + Monthly Forecast

Сборка без отката и без смены визуального направления.

## Что добавлено

- Планирование собрано в одну папку: задачи, календарь, будущие траты, покупки, личные хотелки, хотелки Полины и обязательные платежи Полины.
- У каждого пункта плана есть пометка: `Я`, `Полина`, `Общее`.
- Плановые покупки/траты теперь поддерживают владельца, тип, месяц, ссылку, картинку и включение в прогноз.
- В финансах появился блок “Запланированные расходы по месяцам”.
- В бюджете добавлен месячный план расходов из единой папки планирования.
