# V79 CLEAN RUNTIME

Это аварийное исправление удаляет подключение старых JS/CSS-слоёв. Загружаются только `app-v79-unified.js` и `styles-v79-unified.css`.

Перед загрузкой скачайте экспорт данных. Затем замените `index.html`, `sw.js` и добавьте два файла V79. Старые файлы можно оставить в репозитории: они больше не подключаются.

После commit полностью закройте вкладку и откройте сайт заново. Если браузер держит старый service worker, один раз нажмите `Ctrl+Shift+R`. В настройках V79 есть кнопка очистки только UI-кэша без удаления личных данных.
