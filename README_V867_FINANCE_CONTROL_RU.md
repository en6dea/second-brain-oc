# Second Brain OS V86.7 — лимиты и финансовый контроль

## Новые возможности

### Категории расходов
- Для каждой расходной категории можно установить месячный лимит.
- Для категории задаётся тип по умолчанию: обязательный или необязательный расход.
- Настройки категории можно изменить в отдельной форме.
- Старые операции не удаляются и не сбрасываются.

### Операции
- В форме расхода появилась визуальная плашка выбора:
  - обязательный расход;
  - необязательный расход.
- При выборе дохода эта плашка автоматически скрывается.
- При выборе категории приложение предлагает её тип по умолчанию.
- При превышении лимита финансовый помощник останавливает сохранение и просит отдельное подтверждение.
- При достижении 80% лимита приложение показывает предупреждение.

### Импорт банковского CSV
- Сохранены выпадающие списки «Доход / Расход» и категорий.
- Для расходных строк добавлен выпадающий список обязательности.
- Дубли внутри файла и в существующей базе не импортируются повторно.
- Категории, комментарии и обязательность можно проверить до импорта.

### Подробная аналитика
- доходы;
- все расходы;
- чистый денежный поток;
- норма сбережений;
- обязательные и необязательные расходы;
- доля этих расходов от дохода и общей суммы расходов;
- лимит, расход, остаток и процент использования по каждой категории;
- сравнение категории с предыдущим месяцем;
- категории с превышением и приближением к лимиту;
- движение денег за последние 6 месяцев.

### Финансовый помощник
Помощник сообщает:
- какие лимиты превышены;
- какие категории достигли 80% лимита;
- когда расходы стали выше доходов;
- когда доля необязательных расходов слишком велика;
- какие категории ещё не имеют лимита.

## Установка

1. Сохраните резервную JSON-копию данных в приложении.
2. Замените в корне GitHub-репозитория:
   - `index.html`;
   - `app-v86-direct-20260721.html`.
3. Выполните Commit changes.
4. После публикации откройте приложение напрямую с параметром версии `?v=867#finance`.

`force-update.html` использовать не требуется.
