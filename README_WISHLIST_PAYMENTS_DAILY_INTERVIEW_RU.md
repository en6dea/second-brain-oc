# Second Brain OS — Wishlist + Payments + Daily Interview

Точечная сборка поверх рабочей версии без отката дизайна.

## Что добавлено

- В планировании всё объединено в одну папку: задачи, календарь, покупки, будущие траты, вишлист и обязательные платежи.
- Отдельная категория «Полина» убрана. Теперь у каждого пункта есть поле «Кому принадлежит»: Я, Полина или Общее.
- «Хотелки» переименованы в «Вишлист».
- Платежи объединены по тому же принципу: владелец + месяц + сумма + прогноз.
- Старые хотелки/платежи Полины мигрируют в единое планирование и больше не двоят прогноз.
- Шаблонные пункты из прошлой сборки удаляются автоматически.
- В финансах и бюджете показывается запланированная сумма расходов по месяцам.
- В задачах и планировании картинку теперь можно загрузить с устройства, а не вставлять URL.
- В привычках появился ежедневный блок «Интервью с подсознанием».
- Интервью сохраняется в привычки и в заметки с тегами: рефлексия, подсознание, интервью, цитата.
- В интервью добавлены цитата, вопрос дня и поле рефлексии «как в блокноте».
- В сферах жизни добавлены подпапки по каждой категории.
