# Second Brain OS — Life Next Light Tech

Чистая light-tech сборка с новыми личными направлениями без смены домена.

Добавлено:
- Привычки как отдельное направление: создание, редактирование, удаление, отметки и график повторений.
- Личная жизнь: список фильмов и сериалов с ссылками на трейлеры/страницы.
- Люди: визуальные карточки с фото URL, датами рождения, подарками и полезными ссылками.
- Сферы жизни: плитки вместо выпадающего списка.
- Планирование: отдельные плитки для задач, календаря и поездок.
- Бюджет: график трат по дням, выделение дней с максимальными расходами, анализ дней.
- Импорт банка: автоматическое удаление дублей, ручной фактический остаток, редактирование типа/суммы/категории до сохранения.
- Финансы: возврат долгов в новом дизайне, редактирование операций и категорий.
