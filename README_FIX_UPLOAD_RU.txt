Как загрузить:
1. Распакуйте этот ZIP на компьютере.
2. В GitHub откройте репозиторий en6dea/second-brain-oc и ветку main.
3. Нажмите Add file → Upload files.
4. Перетащите ВСЕ файлы из распакованной папки в корень репозитория.
5. Проверьте, что index.html заменяется и его размер около 696 КБ.
6. Сделайте Commit directly to main.
7. Откройте Settings → Pages.
8. Выберите Source: Deploy from a branch; Branch: main; Folder: /(root).
9. Дождитесь Pages build and deployment.
10. Откройте https://en6dea.github.io/second-brain-oc/?v=88#today и нажмите Ctrl+Shift+R.

Важно:
- Папку .github/workflows пока не загружайте.
- После восстановления QA workflow можно вернуть отдельным коммитом.
- Личные данные приложения этот пакет не очищает.
