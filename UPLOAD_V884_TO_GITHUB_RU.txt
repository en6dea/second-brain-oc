КАК ЗАГРУЗИТЬ V88.4 В GITHUB

1. Перед загрузкой откройте текущую версию приложения.
2. Настройки → Резервная копия JSON → Скачать.
3. Распакуйте SECOND_BRAIN_OS_V88.4_SAFE_FINANCE_FULL.zip.
4. Откройте внутри папку second-brain-oc-v884.
5. Загрузите СОДЕРЖИМОЕ этой папки в корень репозитория en6dea/second-brain-oc.
6. Подтвердите замену index.html, sw.js, pwa-v88.js и sbos-build.txt.
7. Не запускайте reset-*.html.
8. Дождитесь GitHub Pages.
9. Откройте приложение с параметром ?v=884 либо нажмите Ctrl + Shift + R.
10. Проверьте надпись внизу: V88.4 · SAFE FINANCE.
11. Финансы → Счета и резервы → Сверить остатки.

Если старая версия осталась в PWA:
- откройте приложение в обычной вкладке браузера;
- выполните Ctrl + Shift + R;
- затем снова запустите установленное приложение.

Личные данные при обновлении не должны очищаться. При первом запуске V88.4 создаёт автоматическую копию состояния до миграции.
