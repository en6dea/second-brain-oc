# Second Brain OS — Folder Drilldown + Debt Date Final

Сборка без отката и без смены дизайна.

## Что изменено

- Основные разделы переведены в режим «папка → подпапки → отдельный лист».
- В разделе «Сферы жизни» больше не показываются сразу все категории и блоки на одной перегруженной странице.
- При входе в «Сферы жизни» видны только основные папки: Люди, Заметки, Планирование, Привычки, Книги, Идеи, Личная жизнь, Документы, Путешествия, Дневник, Вишлист, Платежи.
- При открытии основной папки показываются её подпапки.
- При открытии подпапки показываются только её записи.
- Аналогично разбиты Планирование, Финансы, Бюджет, Цели, Привычки, Долги, Путешествия, Импорт и Диагностика.
- Для долгов усилено редактирование даты: поле «Когда отдать / когда должны вернуть» сохраняется и для обычных долгов, и для долгов, определённых по банковским операциям.
- Ежедневное «Интервью с подсознанием» доступно через Привычки и Сферы жизни → Дневник.
