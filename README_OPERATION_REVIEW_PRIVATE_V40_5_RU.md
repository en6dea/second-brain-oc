# Second Brain OS — Operation Review Private V40.5

Версия: `v40-5-operation-review-private-20260629`

## Что добавлено

- Новый раздел **Операции** в блоке Деньги.
- Разбор всех банковских операций за выбранный месяц.
- Быстрая смена типа операции: расход / доход.
- Быстрая смена категории прямо в таблице.
- Флаг **не считать в лимит** для переводов, копилки, внутренних движений.
- Расширенное окно операции: дата, сумма, тип, категория, описание, лимит, связь с долгом.
- Возможность привязать операцию к существующему долгу.
- Возможность создать новый долг прямо из операции.
- Возможность пометить операцию как платёж / погашение долга.
- Правила категорий: запомнить правило для похожих операций и применить к похожим.
- Плановые платежи теперь можно отнести к категории и долгу.

## Как пользоваться

1. Загрузить CSV банка через Импорт.
2. Открыть раздел **Операции**.
3. Выбрать месяц.
4. Разнести операции по категориям.
5. Переводы / копилку / внутренние движения отметить как **не в лимит**.
6. Операции, связанные с долгами, открыть и привязать к долгу или создать долг.
