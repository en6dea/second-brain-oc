# Second Brain OS — V41 Force Update Hotfix

База: V40 Finance Cockpit + подготовка Альфа-Банк API.

## Что исправлено

- Исправлено зависание `force-update.html` на экране «Обновление V40».
- Причина: в inline-скрипте страницы обновления была некорректная строка переноса, из-за этого JavaScript не запускался и редирект не происходил.
- Добавлен безопасный fallback: кнопка ручного открытия приложения.
- Обновлены cache-busting build-метки для `index.html`, `sw.js`, `version-check.html`.

## Что не менялось

- Данные пользователя.
- Финансовая логика V40.
- CSV-импорт V39.
- Автосинхронизация и дневник V38.
- Диагностика и бэкапы V37.

## После загрузки

1. Открыть `version-check.html?v=v41`.
2. Нажать очистку кэша.
3. Проверить `#finance`.

Ожидаемый бейдж: `V41 · ФИНАНСЫ · COCKPIT · HOTFIX`.
