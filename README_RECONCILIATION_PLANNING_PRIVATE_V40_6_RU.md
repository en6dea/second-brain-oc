# Second Brain OS — Reconciliation + Planning Private V40.6

Версия V40.6 закрывает две задачи:

1. Сверка банковского CSV и корректный расчёт баланса.
2. Планирование личной жизни по датам с учётом будущих расходов и бюджета месяца.

## Главное

- Банковский баланс считается отдельно от лимита расходов.
- Внутренние переводы, копилка и операции «не в лимит» не ломают лимит, но могут учитываться в банковском движении.
- Добавлен раздел «Сверка»: начальный остаток, фактический остаток, расчётный остаток, расхождение.
- Добавлен журнал импортов CSV.
- Добавлен раздел «Планирование»: события по датам, сумма, категория, статус, учитывать / не учитывать в бюджете.
- Планы отображаются в календаре.
- План можно отметить как оплаченный — будет создана операция расхода.
