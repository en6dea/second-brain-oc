# Second Brain OS V94 · CSV IMPORT RESTORED

Точечный hotfix возвращает загрузку банковской CSV-выписки непосредственно в раздел **«Финансы → Операции»**.

## Что изменено

- В верхнюю панель страницы «Операции» возвращена кнопка **«⇧ Импорт CSV из банка»**.
- Кнопка использует существующий безопасный сценарий импорта:
  1. выбор CSV-файла;
  2. предварительный просмотр строк;
  3. определение доходов, расходов и переводов;
  4. проверка категорий;
  5. автоматическое исключение дублей;
  6. подтверждение импорта.
- Экспорт CSV, переименование CSV-категорий и ручное создание операции сохранены.
- Обновлена версия PWA-кэша, чтобы GitHub Pages не показывал старую панель действий.
- Перед первым запуском V94 создаётся новый резервный снимок; основной ключ данных `secondBrainOS.v1` не менялся.

## Что не изменялось

- структура финансовых операций;
- IndexedDB и localStorage;
- существующие банковские записи;
- категории и правила классификации;
- остальные экраны V93.
