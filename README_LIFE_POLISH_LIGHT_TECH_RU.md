# Second Brain OS — Life Polish Light Tech

Чистая light tech-сборка без смены домена и без старых версионных слоёв.

## Добавлено
- ручной фактический остаток при CSV-импорте;
- редактирование и удаление заметок;
- кликабельные фильтры и действия задач на главной;
- редактирование и удаление сфер жизни;
- кнопки Google Calendar для задач и событий;
- более красивые финансовые плашки;
- график трат по дням в бюджете с выделением крупных дней;
- SMART-цели и рекомендации по принципу 52 недель;
- финансовые принципы без перегруза;
- цитата дня и поле “что я думаю”;
- папки сфер жизни: Люди, Заметки, Планирование, Книги, Идеи.

## Установка
Заменить файлы в корне GitHub Pages:
- index.html
- app.js
- sw.js
- manifest.webmanifest
- offline.html
- force-update.html

Открывать обычный домен: https://en6dea.github.io/second-brain-oc/
Если браузер держит старый кэш: /force-update.html
