# Second Brain OS — Bank Import Private V40.2

Версия: `v40-2-bank-import-private-20260629`

## Что добавлено

- Реальный импорт банковской CSV-выгрузки.
- Поддержан формат файла с колонками:
  - `operationDate`
  - `transactionDate`
  - `accountName`
  - `accountNumber`
  - `cardName`
  - `cardNumber`
  - `merchant`
  - `amount`
  - `currency`
  - `status`
  - `category`
  - `mcc`
  - `type`
  - `comment`
  - `bonusValue`
  - `bonusTitle`
- Предпросмотр перед импортом.
- Автоматическое определение:
  - дата операции;
  - доход / расход;
  - сумма;
  - категория;
  - описание операции;
  - статус операции.
- Защита от повторного импорта дублей.
- Автоматический снимок данных перед импортом.
- Внутренние переводы и копилка помечаются как `не в лимит`, чтобы не ломать расчёт доступных расходов.

## Как пользоваться

1. Перед импортом открой раздел **Данные**.
2. Нажми **Скачать полный бэкап**.
3. Открой раздел **Импорт**.
4. Нажми **Загрузить файл банка**.
5. Выбери CSV-файл банка.
6. Проверь предпросмотр.
7. Нажми **Импортировать новые операции**.
8. Проверь раздел **Финансы**.

## Важно

Файл обрабатывается локально в браузере/приложении. Он не отправляется на сервер.

Если банк отдаёт файл в Excel `.xlsx`, пока лучше сохранить его как CSV и загрузить CSV. Следующим шагом можно добавить XLSX-парсер.
