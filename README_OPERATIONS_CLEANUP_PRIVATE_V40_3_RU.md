# Second Brain OS — Operations Cleanup Private V40.3

Версия: `v40-3-operations-cleanup-private-20260629`

## Что добавлено

- Безопасная очистка операций за выбранный период.
- Можно удалить:
  - все операции за период;
  - только операции, загруженные из банка;
  - только ручные операции.
- Перед удалением автоматически создаётся локальный снимок восстановления.
- Предпросмотр удаления: количество операций, расходы, доходы, операции “не в лимит”, источник.
- Новый режим импорта: **Заменить период этим CSV**.
- При замене приложение удаляет все операции за даты файла и импортирует весь CSV заново.
- Защита от дублей внутри CSV.
- Финансовый дашборд, Data Core, календарь и PWA сохранены.

## Как перезалить месяц

1. Открой **Данные**.
2. Нажми **Скачать полный бэкап**.
3. Открой **Импорт**.
4. Нажми **Загрузить файл банка**.
5. Выбери CSV.
6. В предпросмотре нажми **Заменить период этим CSV**.
7. Проверь **Финансы** и категории.

## Важно

Перед массовым удалением всегда скачивай полный бэкап. Дополнительно приложение создаёт локальный снимок восстановления автоматически.
