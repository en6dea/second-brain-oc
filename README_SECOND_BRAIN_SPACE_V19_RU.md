# Second Brain OS — V19 Second Brain Space

Что реализовано:
- Единое пространство «второго мозга» в стиле Notion: папки, карточки-базы, быстрый захват.
- Красивые цветные иконки для основных разделов.
- Долги вынесены отдельной папкой рядом с финансами.
- В долгах разделено: «Я должен» и «Мне должны».
- Для долгов есть сумма, срок, статус, комментарий, напоминание, редактирование, закрытие и удаление.
- Привычки оформлены как таблица по дням: дни читаются сверху, ячейки кликабельные, есть прогресс и стрики.
- Покупки и желания поддерживают дату, сумму, URL, URL картинки, флаг «учитывать в бюджете».
- Путешествия остались внутри сфер жизни.
- «Хотелки» как отдельная папка не используются; старые хотелки при наличии переносятся в желания.
- Все записи основных разделов можно добавлять, редактировать и удалять.
- Папки в левом меню можно переименовывать и скрывать, восстановление через профиль.
