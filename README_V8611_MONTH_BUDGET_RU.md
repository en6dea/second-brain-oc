# Second Brain OS V86.11 — бюджет будущего месяца

## Новое

- Ожидаемый доход для каждого месяца.
- День зарплаты.
- Резерв или сумма накоплений, которую нельзя распределять.
- Автоматическое распределение лимитов категорий от ожидаемого дохода.
- Отдельные лимиты для каждого месяца с сохранением базовых лимитов категорий.
- Плановые расходы учитываются при формировании лимитов и прогноза остатка.
- Лимиты можно менять прямо в аналитике и на странице планирования.
- Карточки обязательных, необязательных расходов и лимитов стали кликабельными.
- После настройки категории приложение возвращает на прежнее место списка.
- Удалённые категории переносятся в архив и исчезают из выпадающих списков.
- Добавлен третий тип операции: «Перевод / нейтральное движение».
- Переводы между счетами, возвраты и прочие нейтральные движения не считаются доходом или расходом.
- Импорт и экспорт CSV поддерживают новый тип операции.

## Установка

Замените в корне репозитория:

- `index.html`
- `app-v86-direct-20260721.html`

После публикации откройте обычную ссылку приложения с параметром `?v=8611` и выполните один раз `Ctrl + Shift + R`.

Данные не очищаются. Перед обновлением подготовлена резервная копия V86.10.
