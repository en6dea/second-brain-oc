# Second Brain OS V41 — Import Review Core

Что изменено:

- Добавлен финальный слой проверки импорта перед сохранением.
- В каждой строке импорта появилась выпадающая категория.
- Можно вписать новую категорию вручную.
- Можно удалить отдельную строку из импорта до сохранения.
- Можно массово заменить категорию.
- Можно указать начальный баланс периода и фактический баланс сейчас прямо в окне импорта.
- Импорт сохраняет проверенные строки, пишет журнал импорта и страховочную копию.
- Добавлен action bridge для новых кнопок, чтобы они работали через текущий V40.12 hardwire-слой.
