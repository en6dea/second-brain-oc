# Second Brain OS V46.4 — Budget Coach UI Safe

Безопасная визуальная доработка раздела **Бюджет** поверх стабильной V46.3.

## Что изменено

- новый красивый экран бюджета;
- блок **Деньги до конца месяца**;
- дневной остаток по лимитам;
- конверты по категориям;
- статусы: норма / риск / перерасход / нужен лимит;
- карточка “Что сделать сейчас”;
- структура: база / желания / будущее / долги;
- мягкий version guard против мигания старой плашки.

## Что не менялось

- ядро приложения;
- импорт;
- структура данных;
- финансы;
- дорожная карта;
- синхронизация.
