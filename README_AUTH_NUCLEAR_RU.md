# AUTH NUCLEAR v2

Исправляет ситуацию, когда GitHub Pages/PWA продолжает видеть старый пустой firebase-config.js.

Заменить/загрузить в КОРЕНЬ репозитория все файлы из архива.
Важно: не загружать папку, а именно файлы внутрь репозитория рядом с index.html.

Проверка после деплоя:
https://en6dea.github.io/second-brain-oc/sb-firebase-config.nuclear.js?v=auth-nuclear-v2-20260618

Там должен быть реальный apiKey, а не PASTE_API_KEY_HERE.

Открывать приложение:
https://en6dea.github.io/second-brain-oc/?v=auth-nuclear-v2-20260618

На экране снизу слева появится маленькая плашка AUTH v2 — это значит, что новый index.html реально подгрузился.
