# V72 — «Состояние Полины»

Сборка: `second-brain-space-v72-polina-state-20260716-r2`

Добавлен новый раздел **«Состояние Полины»** в папку:

`Личная память → Я и люди`

## Что умеет раздел

- календарь по месяцам;
- состояния: **Отличное**, **Хорошее**, **Нейтральное**;
- комментарий к каждому дню;
- отдельные отметки **начала** и **конца** месячных, которые можно сочетать с состоянием дня;
- цветные плашки дней по состоянию;
- фактические дни месячных и ориентировочный прогноз выделяются отдельно;
- прогноз следующего цикла;
- после нескольких отметок начала прогноз рассчитывается по среднему фактических циклов;
- длительность месячных уточняется по отметкам начала и конца;
- история всех состояний сгруппирована по месяцам;
- данные сохраняются в `state.polinaDays`, входят в JSON-backup и уже поддерживаются облачной синхронизацией V67.

## Важно

Прогноз цикла является ориентировочным инструментом личного планирования и не заменяет медицинские рекомендации.
