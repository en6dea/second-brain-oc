# Исправление цикла force-update для V86

Проблема: прежний force-update.html автоматически перенаправлял текущую вкладку. Старый Service Worker мог продолжать управлять этой вкладкой и снова отдавать force-update.html, из-за чего возникал бесконечный цикл обновления.

Исправление:
- автоматическое перенаправление удалено;
- кэш и регистрации Service Worker очищаются;
- приложение открывается кнопкой в новой вкладке;
- новая вкладка уже не должна контролироваться старым Service Worker;
- localStorage, IndexedDB и пользовательские данные не очищаются.

Установка:
1. Замените force-update.html в корне репозитория.
2. Сделайте Commit changes.
3. После публикации снова откройте /force-update.html.
4. Нажмите «Открыть V86 в новой вкладке».
