# Second Brain OS V47 — Life Categories Core

Версия V47 расширяет приложение из финансового трекера в личный Second Brain OS.

## Главное

- Новый раздел **Жизнь**.
- Сферы жизни: финансы, здоровье, дом, машина, документы, поездки, обучение, привычки, идеи, питомцы.
- Новая универсальная сущность `lifeEntries` для записей жизни.
- Типы записей: задача, заметка, расход, доход, цель, событие, документ, привычка, идея, имущество.
- Универсальный быстрый ввод V47: обычный текст раскладывается по сферам жизни.
- Связь с финансами: расход/доход из записи автоматически создаёт операцию.
- Связь с задачами, целями, привычками и заметками.
- Карточка “Второй мозг жизни” на главном экране.
- Диагностика V47 проверяет личное ядро и старую плашку версии.
