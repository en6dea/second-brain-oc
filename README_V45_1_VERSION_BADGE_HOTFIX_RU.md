# Second Brain OS V45.1 — Version Badge Hotfix

Точечный hotfix поверх V45 Analytics Pro + Budget Coach.

## Что исправлено

- Исправлена старая плашка версии слева снизу: теперь принудительно отображается `V45.1`.
- Добавлен жёсткий `MutationObserver`, который следит за DOM и не даёт старым слоям V40/V41/V42/V43/V44 перезаписать видимую версию.
- Обновляется `releaseBadge`, `sidebar-status`, `v403-status`, `data-build-version`, `document.title` и meta `second-brain-build`.
- В диагностике добавлена карточка "Версия интерфейса" с кнопкой "Исправить плашку версии".
- Обновлён service worker cache name, чтобы GitHub Pages и браузер не держали старую сборку.

## Что не менялось

- Данные пользователя не удаляются.
- Логика бюджета, аналитики, наставника, импорта и категорий не переписывалась.
- Это не новая продуктовая версия, а стабилизационный hotfix для визуальной версии.

## Как открыть после загрузки

```text
https://en6dea.github.io/second-brain-oc/?v=v45-1-version-badge-hotfix-private-20260630&hard=1
```
