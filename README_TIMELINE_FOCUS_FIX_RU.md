# Second Brain OS — Timeline Focus Fix

Сборка сохраняет все текущие категории и clean light-tech дизайн, но исправляет найденные при инспекции места.

## Что исправлено

- Расхождение по балансу теперь раскрывается: факт, расчёт, доходы, расходы, плановые покупки и долги.
- Подушка безопасности объясняет, откуда берётся процент, и редактируется отдельно от целей.
- Рефлексия стала отдельной красивой карточкой и сохраняется в заметки с тегом `рефлексия`.
- Матрица Эйзенхауэра стала кликабельной: задачи можно редактировать прямо из квадрантов.
- В задачи добавлено поле картинки URL, изображение видно в списке.
- Сферы жизни больше не дублируют одни и те же папки на экране.
- Планирование считает только открытые задачи, а не смешивает всё в непонятную цифру.
- Долги возвращены отдельным разделом и сохранены в финансах.
- Привычки оставлены в красивом графическом формате с повторениями.
- Добавлены отдельные страницы: Привычки, Долги, Путешествия.
- Сохранены категории: Обзор, Финансы, Бюджет, Цели SMART, Планирование, Сферы жизни, Привычки, Долги, Путешествия, Люди, Книги, Идеи, Документы.
