# Second Brain OS — V25

Фикс по замечаниям:
- папки планирования Фокус дня / Папки сфер / Приоритеты / Сроки показывают разные данные;
- покупки редактируются и удаляются, legacy plannedPurchases больше не возвращает удалённые карточки;
- дубли покупок удаляются автоматически по названию и сумме;
- идеи больше не возвращаются после удаления пустого списка;
- сайдбар сохраняет прокрутку и нижние папки не прыгают;
- финансы показывают объяснение источников дохода;
- прогноз учитывает ближайшие плановые покупки до конца выбранного периода.
