ВАЖНО: НЕ загружайте ZIP в репозиторий как один файл.

1. Распакуйте архив.
2. В корне репозитория откройте существующий index.html и нажмите Delete file либо замените его файлом из этого архива.
3. Загрузите index.html и force-update.html именно в корень репозитория, рядом с README.md.
4. Сделайте Commit changes.
5. Проверьте, что размер index.html на GitHub около 270 КБ, а не 21 КБ.
6. Откройте /force-update.html один раз.

Личные данные не очищаются.
