# Second Brain OS — Folder System V2

Сборка без отката. Исправляет структуру только для двух разделов: Планирование и Сферы жизни.

## Логика

- Планирование открывается как папка. Внутри только папки: Задачи, Календарь, Вишлист, Платежи, Будущие траты, Покупки.
- Каждая папка открывает второй уровень подпапок.
- Только после выбора подпапки открывается отдельный лист с содержимым.
- Сферы жизни работают так же: сначала папки сфер, потом подпапки, потом отдельный лист.
- Остальные разделы не дробятся.
- Обзор оставлен как главный экран со сводкой, но без длинной простыни папок.
