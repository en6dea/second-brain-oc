# Second Brain OS — V40.10 Button Runtime + Balance Core Private

Сборка: `v40-10-button-runtime-balance-private-20260629`

## Что исправлено

- Усилен runtime-обработчик кликов по всем `data-action` и `data-page`.
- Добавлена защита от двойного срабатывания кнопок.
- Добавлен fallback-сейв для критичных модалок: расход, доход, платежи, планы личной жизни.
- Ошибки кнопок теперь пишутся в диагностику, а не выглядят как "кнопка не работает".
- В разделе Данные добавлен блок Runtime кнопок V40.10.
- В Финансах добавлена быстрая кнопка перехода к фактическому остатку по картам.
- Сохранение фактических остатков по картам/счетам сохранено из V40.9.

## Как проверять

1. Открыть `?v=v40-10-button-runtime-balance-private-20260629`.
2. Данные → Скачать полный бэкап.
3. Данные → Проверить кнопки.
4. Финансы → Расход → сохранить.
5. Планирование → создать план → сохранить.
6. Сверка → факт по картам → сохранить факт.
