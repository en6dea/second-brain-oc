# Безопасность данных

Second Brain OS обрабатывает чувствительные личные сведения. По умолчанию данные сохраняются локально в `localStorage` текущего браузера.

## Никогда не добавляйте в публичный репозиторий

- банковские выгрузки и финансовые отчёты;
- базы клиентов и контактов;
- резервные копии приложения;
- API-ключи, токены GitHub/Firebase и пароли;
- документы, медицинские сведения и фотографии без осознанного согласия владельца.

`.gitignore` блокирует типовые приватные файлы, но не заменяет проверку перед коммитом. Если секрет уже попал в Git, одного удаления файла недостаточно: отзовите секрет и очистите историю репозитория.

## Рекомендации для версии «для других»

- отделить персональные данные каждого пользователя через авторизацию;
- шифровать чувствительные данные при хранении и передаче;
- добавить понятный экспорт и полное удаление аккаунта;
- собирать только необходимые данные и публиковать политику конфиденциальности;
- пройти отдельный аудит безопасности перед реальным запуском.
