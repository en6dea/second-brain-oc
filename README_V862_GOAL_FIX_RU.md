# Second Brain OS V86.2 — исправление мастера целей

Исправлена остановка мастера на шаге 6 из 7.

Причина: на финальном экране вызывалась отсутствующая функция подписи типа цели `goalTypeLabel`, из-за чего JavaScript завершался ошибкой и визуально казалось, что кнопка «Продолжить» не работает.

Изменения:
- добавлена функция подписи типов целей;
- кнопкам мастера добавлен `type="button"`;
- переходы «Продолжить» и «Назад» защищены обработкой ошибок;
- сохранены все существующие данные и фирменный дизайн.

Замените `index.html` и `app-v86-direct-20260721.html` в корне репозитория.
