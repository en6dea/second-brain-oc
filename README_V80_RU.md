# Second Brain OS V80

## Главное
- Крупнее и легче шрифт во всём приложении.
- Симметричные карточки привычек, добавление и удаление с архивной копией.
- Каждый день новый набор из 3 вопросов интервью с подсознанием.
- Интервью больше не попадает в заметки и личную память; старые ошибочные записи автоматически перенесены в правильный раздел с резервной копией.
- Удаление операций с автоматической копией в архиве.
- Анализ категорий распознаёт старые значения типов операций: «Доход», «Расход», income/expense и отрицательные суммы.
- Люди получили полноценные анкеты: фото, день рождения, напоминание, контакты, интересы, границы, вишлист и идеи подарков.
- Желания, книги, фильмы и путешествия получили отдельный URL изображения и прямую ссылку.

Перед обновлением скачайте JSON-экспорт. V80 также автоматически создаёт снимок V79 в IndexedDB.
