# Second Brain OS — V40.9 Data Guard + Balance Anchor Private

Цель версии: защитить данные операций при обновлениях и сделать остаточный баланс устойчивым.

## Что добавлено

- Data Guard: страховочная копия состояния в localStorage.
- Автовосстановление операций, если после обновления приложение открылось с демо-операциями или пустой базой.
- Кнопка «Страховка перед обновлением» в разделе Данные.
- Кнопка «Восстановить операции из страховки».
- Фиксация фактического остатка по картам / счетам.
- Таблица «Несгораемый остаток» в Сверке.
- Перенос факта текущего месяца в начальный остаток следующего месяца.

## Как правильно считать остаточный баланс

1. Загрузи CSV банка.
2. В Сверке введи фактические остатки по картам.
3. Нажми «Сохранить факт по картам».
4. Нажми «Зафиксировать текущий факт».
5. В конце месяца нажми «Перенести факт в начало следующего месяца».

Формула:

начальный остаток + движение по CSV = расчётный остаток.

Фактический остаток по карте сравнивается с расчётным.

## Перед каждым обновлением

Данные → Страховка перед обновлением.

Это создаёт локальный снимок, обновляет страховочную копию и скачивает JSON-бэкап.
