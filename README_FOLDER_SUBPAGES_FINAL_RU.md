# Second Brain OS — Folder Subpages Final

Сборка без отката и без смены дизайна. Главная логика: основная папка открывает подпапки, а каждая подпапка открывается отдельной страницей, чтобы не искать нужное на перегруженной ленте.

## Что добавлено

- Планирование стало структурой: папка → подпапки → отдельная страница.
- Подпапки планирования:
  - Задачи
  - Календарь
  - Вишлист
  - Платежи
  - Будущие траты
  - Покупки
  - Все пункты
- В каждой подпапке показываются только её записи.
- Вишлист и платежи остаются общими, с полем владельца: Я, Полина, Общее.
- Сферы жизни тоже стали структурой: папка → подпапки → отдельная страница.
- Убрана перегрузка: при входе в Сферы жизни сначала видны папки, потом подпапки, потом содержимое.
- Сохранены текущие данные и текущий light-tech / Timeline Focus дизайн.

## После загрузки

1. Открыть `force-update.html`
2. Потом открыть обычный домен приложения
3. Проверить: Планирование → Вишлист / Платежи / Будущие траты
4. Проверить: Сферы жизни → любая папка → подпапка
