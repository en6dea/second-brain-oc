# Second Brain OS — V40.7 Lifestyle Planning + Auto Reconcile Private

Версия для личного приложения перед десктопной сборкой.

## Что добавлено

- Планирование поездок, отдыха, развлечений, встреч и подарков по датам.
- Раздел `Планирование` теперь находится в группе `Люди`.
- План можно связать с конкретным человеком.
- У плана есть тип: Поездка / Отдых / Развлечение / Встреча / Подарок / Семья / Другое.
- План можно включить или не включить в бюджет месяца.
- План со статусом `Оплачено` создаёт операцию расхода и попадает в финансы.
- План виден в календаре на выбранную дату.
- Добавлены категории: Личная жизнь, Поездки, Отдых, Встречи, Семья.

## Сверка банка

- Для банковской сверки теперь используется `operationDate` как дата проводки, а не `transactionDate`.
- В предпросмотре CSV можно сразу ввести начальный остаток месяца и фактический остаток после выгрузки.
- После импорта раздел `Сверка` автоматически показывает расчётный остаток и расхождение.
- Все операции банка учитываются в банковском балансе, даже если они не входят в лимит расходов.
- Операции `в обработке` отделяются от выполненных.
- Журнал импортов сохраняет период, количество операций, режим импорта и контрольные суммы.

## Ежедневный режим

1. Скачать свежий CSV из банка.
2. Открыть `Импорт`.
3. Загрузить CSV.
4. Ввести фактический остаток из банка на момент выгрузки.
5. Нажать `Импортировать только новые` или `Обновить / заменить период этим CSV`.
6. Открыть `Сверка` и проверить расхождение.
7. Открыть `Операции` и разнести категории/долги.

## Проверка после загрузки

Открыть:

`https://en6dea.github.io/second-brain-oc/?v=v40-7-lifestyle-planning-private-20260629`

Проверить:

- `Импорт → загрузка CSV → поля остатков`
- `Сверка → расчётный остаток / расхождение`
- `Календарь → планы по датам`
- `Операции → категория / долг / в лимит / в баланс`
