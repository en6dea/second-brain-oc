## Что заменить на GitHub

Заменить файлы в корне репозитория:

```text
app.js
index.html
sw.js
force-update.html
version-check.html
README_SECOND_BRAIN_SPACE_V44_RU.md
```

## Проверка

1. Открыть `version-check.html?v=v44`.
2. Открыть `force-update.html?v=v44-system-core`.
3. Проверить бейдж внизу: `V44 · SYSTEM CORE · ПОИСК + ВНИМАНИЕ`.
4. Проверить страницы:
   - `#attention`
   - `#global-search`
   - `#expense-review`
   - `#diagnostics`
