## Как поставить

1. Распаковать ZIP.
2. Залить всё содержимое в корень GitHub Pages репозитория.
3. Открыть:

```txt
https://en6dea.github.io/second-brain-oc/?v=v40-1-qa-polish-private-20260626
```

4. В разделе “Данные” скачать полный бэкап.
5. Проверить календарь, финансы, долги, платежи и быстрый ввод.
