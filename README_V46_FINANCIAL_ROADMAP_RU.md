# Second Brain OS V46 — Financial Roadmap

Версия V46 развивает V45.1 и добавляет личную финансовую дорожную карту без CRM/клиентов.

## Главное

- Новый раздел **Дорожная карта**.
- Подушка безопасности: цель на 1–12 месяцев базовых расходов.
- Долги: сумма активных долгов и задача на первый шаг погашения.
- Накопления: правило “заплати себе сначала” и задача на 10% дохода.
- Категории: напоминание разобрать операции без категории.
- Бюджет: связь с V45 Budget Coach.
- Диагностика V46: проверка индекса пути, подушки, долгов и версии.
- Жёсткое исправление видимой версии: V46 перекрывает старую плашку V45.1.
