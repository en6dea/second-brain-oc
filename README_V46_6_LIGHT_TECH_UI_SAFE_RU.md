Название приложения теперь стабильно: **Second Brain OS**. Версия больше не является названием продукта и показывается только как маленькая техническая сборка.

- чистая светлая Apple/light-tech эстетика;
- белый фон, мягкие серые поверхности, синие акценты;

- аккуратный sidebar;

- более лёгкая мобильная навигация;
- закреплено название вкладки браузера: `Second Brain OS`;

- версия технически: `V46.6 LIGHT TECH UI`.

- финансы;

- дорожная карта;
- localStorage;
