# Second Brain OS v28 — Calendar Space

Добавлена отдельная папка **Календарь** в группе **Пространство**, над папкой **Задачи**.

Что добавлено:
- календарь событий и напоминаний;
- календарь показывает задачи, события, долговые напоминания и плановые покупки;
- добавление событий по сферам;
- отображение нагрузки по дням и сферам;
- кнопка добавления события в Google Calendar;
- редактирование и удаление событий;
- текущий дизайн приложения сохранён.

Сборка: `second-brain-space-v28-calendar-events-20260701`
