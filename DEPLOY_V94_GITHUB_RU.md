# Публикация Second Brain OS V94 в GitHub

1. Скачайте архив `second-brain-oc-v94-upload-files.zip`.
2. Распакуйте его.
3. Загрузите всё содержимое в корень репозитория `en6dea/second-brain-oc` с заменой файлов.
4. Дождитесь завершения GitHub Pages.
5. Откройте:

`https://en6dea.github.io/second-brain-oc/?v=94#finance-operations`

## Проверка

На странице «Операции» в верхней панели должны быть кнопки:

- Категории;
- **Импорт CSV из банка**;
- Переименовать CSV;
- Экспорт;
- Операция.

Нажмите «Импорт CSV из банка». Должно открыться окно выбора файла. После выбора CSV должен появиться предпросмотр перед добавлением операций.

В левом нижнем углу должна отображаться метка `V94 · CSV IMPORT RESTORED`.
