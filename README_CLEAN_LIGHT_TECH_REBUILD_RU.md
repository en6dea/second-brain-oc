# Second Brain OS — Clean Light Tech Rebuild

Полная визуальная пересборка интерфейса в стиле «Вариант 2 — Чистый light tech».

## Главное
- Название приложения всегда: `Second Brain OS`.
- Домен не меняется от версии к версии.
- Версия не выводится как название продукта.
- Старые V40/V41/V46-слои больше не прогружаются внутри интерфейса.
- `sw.js` очищает старые кэши Second Brain OS.
- Данные сохраняются в прежнем ключе localStorage: `secondBrainOS.v1`.
