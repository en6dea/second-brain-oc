# Second Brain OS V37 — Stability Kit

Стабильная надстройка поверх V36 без отката дизайна и данных.

Добавлено:
- отдельная папка **Диагностика** в сервисном блоке;
- проверка версии, service worker, размера localStorage и структуры данных;
- резервная копия: скачать JSON;
- восстановление данных из JSON-файла;
- понятный блок синхронизации устройств через GitHub Gist;
- визуальный полиш карточек, пустых состояний и hover-эффектов;
- финансы: план недели, лимит дня, сценарии закрытия долгов;
- календарь: цветовая нагрузка недели и распределение по сферам.

Сборка: `second-brain-space-v37-stability-kit-20260706`
