# Second Brain OS — Modal Actions Fix Private V40.4

## Что исправлено

Главный фикс: кнопки внутри модальных окон снова кликабельны.

Причина бага V40.3: модальное окно останавливало всплытие кликов, а общий обработчик `data-action` висел на `document.body`. Из-за этого кнопки в предпросмотре банковского импорта выглядели активными, но действие не запускалось.

## Исправленные сценарии

- Импортировать только новые
- Заменить период этим CSV
- Удалить период вручную
- Другой файл
