SECOND BRAIN OS V88 — КАК ЗАГРУЗИТЬ В GITHUB
================================================

Важно
------
Архив нужно сначала распаковать. Сам ZIP в репозиторий загружать не надо.
Файл data.json и личные данные в этот архив не входят и не изменяются.
Старый styles-v87-obsidian.css не удаляйте: новый V88 использует его как базовый слой.

Пошагово
--------
1. Распакуйте архив second-brain-oc-v88-github-upload.zip.
2. Откройте репозиторий:
   https://github.com/en6dea/second-brain-oc
3. Переключитесь на ветку main.
4. Нажмите Add file → Upload files.
5. Откройте распакованную папку second-brain-oc-v88-github-upload.
6. Перетащите в окно GitHub ВСЁ содержимое этой папки, а не саму внешнюю папку.
   Папку .github тоже перетащите целиком — внутри неё лежит автоматическая QA-проверка.
7. GitHub покажет, что index.html, sw.js, manifest.webmanifest и sbos-build.txt заменяются,
   а остальные файлы V88 добавляются. Это правильно.
8. В поле Commit changes напишите:
   Apply Second Brain OS V88 visual system
9. Выберите Commit directly to the main branch и нажмите Commit changes.
10. Откройте вкладку Actions и дождитесь зелёной проверки
    “Second Brain V88 visual QA”.
11. Подождите 1–3 минуты, затем откройте:
    https://en6dea.github.io/second-brain-oc/?v=88#gamelife
12. Нажмите Ctrl+F5. Если приложение было установлено как PWA — откройте его ещё раз
    и нажмите «Обновить» в появившейся плашке.

Что должно появиться
--------------------
- Значок темы в правой части верхней панели.
- Светлая, тёмная и системная темы.
- Три режима движения: живое, спокойное, выключенное.
- Единые фирменные SVG-иконки в навигации и основных системных блоках.
- Живая карта на главном экране, связанная с реальными целями, привычками и задачами.
- Исправленное нижнее меню и FAB на телефоне.
- V88 · CONSTELLATION в индикаторе версии.

Если GitHub показывает конфликт имён
------------------------------------
Выберите Replace для четырёх существующих файлов:
- index.html
- sw.js
- manifest.webmanifest
- sbos-build.txt

Остальные файлы должны добавиться как новые.

Откат
-----
Если понадобится вернуть предыдущую версию:
1. Откройте страницу Commits репозитория.
2. Найдите коммит “Apply Second Brain OS V88 visual system”.
3. Нажмите Revert и подтвердите новый коммит.
