# Second Brain OS V38 — Автосинхронизация + Дневник подсознания

База: стабильная V37.

## Что добавлено

### 1. Автоматическая синхронизация устройств
После настройки GitHub Gist приложение:
- автоматически проверяет облако при входе;
- проверяет облако при возвращении на вкладку;
- периодически сверяет данные;
- сохраняет изменения в облако после локального сохранения;
- показывает статус синхронизации в «Личное».

Важно: для настоящей синхронизации между разными компьютерами нужно один раз указать один и тот же Gist ID и GitHub token на каждом устройстве.

### 2. Дневник подсознания
В разделе «Личное» добавлена папка «Дневник подсознания».
Внутри:
- ежедневный отклик себе;
- разные вопросы по дням;
- состояние 1–10;
- энергия 1–10;
- сигнал тела;
- главный вывод дня;
- маленькое обещание себе;
- история записей;
- стрик и среднее состояние.
