# Second Brain OS V43 — Personal OS

Версия V43 убирает CRM/клиентский слой из пользовательского интерфейса и переводит приложение в личный режим Second Brain OS.

## Что изменено

- Клиенты убраны из меню, мобильной навигации, dashboard и быстрого ввода.
- Главный экран теперь сфокусирован на личных финансах, импорте, задачах, целях и качестве данных.
- Добавлен блок «План / баланс месяца»: план, начальный баланс, фактический баланс, расхождение.
- Центр категорий очищен от CRM-типов и показывает финансы, задачи, цели, импорт и личные области.
- Быстрый ввод создаёт операции или личные задачи, но не создаёт клиентов.
- Импорт усилен верхним мастером V43: файл, колонки, категории, баланс, сохранение.
- Диагностика проверяет личный режим и показывает, видна ли CRM в интерфейсе.
- Обновлён service worker cache: `second-brain-os-v43-personal-os-private-20260629`.
