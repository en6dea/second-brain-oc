# Публикация Second Brain OS V91 в GitHub Pages

## Вариант 1 — загрузить только обновление

1. Распакуйте `second-brain-oc-v91-upload-files.zip`.
2. Откройте репозиторий `en6dea/second-brain-oc` в GitHub.
3. Загрузите содержимое архива в корень репозитория.
4. Подтвердите замену существующих файлов.
5. Дождитесь завершения GitHub Pages.
6. Откройте приложение по адресу с параметром `?v=91#today`.

## Вариант 2 — заменить репозиторий полной сборкой

Используйте `second-brain-oc-v91-github-ready.zip`. Внутри находится полный каталог проекта.

## Как убедиться, что открылась новая версия

В левом нижнем углу должна отображаться метка:

`V91 · FOCUSED HOTFIX`

## Если браузер показывает старую версию

1. Откройте приложение с `?v=91#today`.
2. Выполните один жёсткий перезапуск страницы: `Ctrl + F5`.
3. При установленном PWA полностью закройте окно приложения и откройте заново.

Новая версия Service Worker использует отдельный кэш `second-brain-os-v91-focused-hotfix-r1` и удаляет устаревшие кэши интерфейса. Пользовательские данные при этом не очищаются.
