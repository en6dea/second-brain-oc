# Публикация Second Brain OS V95 в GitHub

## Рекомендуемый способ

1. Скачайте архив `second-brain-oc-v95-upload-files.zip`.
2. Распакуйте его на компьютере.
3. Откройте репозиторий `en6dea/second-brain-oc` на GitHub.
4. Загрузите содержимое архива в корень репозитория.
5. Подтвердите замену одноимённых файлов.
6. Дождитесь завершения GitHub Pages.
7. Откройте:

`https://en6dea.github.io/second-brain-oc/?v=95#finance`

## Что проверить после публикации

- Внизу отображается `V95 · FINANCE ACTIVE`.
- «Финансы» открываются без сообщения `tips.unshift is not a function`.
- Вкладка «Операции» открывает список операций.
- Кнопка «Импорт CSV из банка» открывает окно выбора файла.
- Существующие операции, счета и долги остались на месте.

Если браузер показывает старую версию, закройте вкладку приложения, откройте URL с `?v=95`, затем один раз выполните жёсткое обновление страницы.
