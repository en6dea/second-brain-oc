# Second Brain OS V77 — постепенная дисциплина

## Что изменено

- Добавлен главный экран `Сегодня` с одной активной привычкой: чтение 20 минут.
- Добавлена минимальная версия привычки на 5 минут для трудных дней.
- Добавлен таймер, фиксация времени, сопротивления началу и короткого наблюдения.
- Добавлен 42-дневный измеряемый эксперимент и недельный разбор.
- Добавлена ежедневная финансовая проверка на 2 минуты.
- Добавлены подсказки при пропусках и бездействии: без стыда и без обнуления прогресса.
- Необязательные разделы собраны на одной странице `Библиотека` с поиском.
- Навигация упрощена: сначала `Сегодня`, `Финансы`, `Привычка`, `Обязательства`.
- Основной шрифт заменён на Onest, снижена чрезмерная жирность текста.

## Сохранность данных

V77 не очищает и не пересоздаёт существующие массивы. Новый модуль записывает только отдельный объект `state.discipline` и использует существующие `settings.currentBalance`, долги и покупки для отображения финансов.

Старые заметки, люди, желания, финансы, трейдинг, пароли, календарь Полины и остальные данные остаются в прежних коллекциях. Они только сгруппированы в новой навигации.

## Установка

Загрузите в корень репозитория файлы из архива обновления и подтвердите замену `index.html` и `sw.js`. Файлы `app-v77-discipline.js` и `styles-v77-discipline.css` будут новыми.

После Commit Changes закройте вкладку приложения, откройте её заново и выполните жёсткое обновление `Ctrl + Shift + R`.

## Откат

Для отката используйте архив `second-brain-oc-backup-v76-before-discipline.zip`: замените содержимое репозитория файлами из него. Дополнительный объект `discipline` в данных не мешает старой версии и будет просто игнорироваться.
