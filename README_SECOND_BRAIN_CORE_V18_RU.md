# Second Brain OS — V18 Second Brain Core

## Что изменено

- Сборка переведена в логику «второго мозга»: быстрый сбор, папки, память, финансы, планирование, привычки и желания.
- «Хотелки» удалены как отдельный раздел. Данные из `wants` при первом запуске переносятся в «Желания», чтобы идеи не потерялись.
- «Долги» вынесены отдельным пунктом меню рядом с «Финансы».
- В долгах есть два отдельных блока: «Я должен» и «Мне должны».
- «Путешествия» убраны из верхнего меню и остаются внутри «Сферы жизни».
- Планирование очищено: Фокус дня, Папки сфер, Приоритеты, Сроки, Покупки, Желания.
- Покупки и желания поддерживают дату, URL, URL картинки, сумму, важность и флаг «в бюджете / вне бюджета».
- Привычки переоформлены в красивый трекер с цветными карточками и отметками за 14 дней.
- Все основные сущности имеют редактирование и удаление.
- Импорт и диагностика спрятаны в профиль/шестерёнку.
- Добавлен видимый бейдж версии V18.
