# Second Brain OS V83.3 — финальные файлы для GitHub

## Добавлено в этой версии

### Книги
- отдельная красивая страница библиотеки;
- статус книги: хочу прочитать, читаю, на паузе, прочитано;
- текущая страница и общее количество страниц;
- визуальный процент прогресса;
- дата начала и дата завершения;
- оценка книги из 10;
- главные мысли и конспект;
- отдельное добавление цитат;
- к цитате можно указать страницу, дату и личный комментарий;
- последние цитаты показываются общей лентой;
- старое поле с цитатами не удаляется и продолжает отображаться.

### Цели
- цель можно удалить непосредственно из карточки;
- цель можно удалить на отдельной странице цели;
- завершённые цели также можно удалить;
- перед удалением создаётся архивная копия;
- связанные задачи не удаляются: у них только снимается связь с удалённой целью.

### Сохранено из V83.2
- импорт банковского CSV;
- удаление дублей;
- редактирование категорий и комментариев;
- прогноз цикла Полины;
- светлый премиальный дизайн и оживлённый GameLife.

## Установка
1. Скачайте резервную копию JSON из приложения.
2. Замените файлы в корне репозитория файлами из пакета обновления.
3. Выполните Commit changes.
4. После публикации один раз откройте `force-update.html`.
