# Second Brain OS — Life Folders + Calendar Light Tech

Чистая light tech-сборка с сохранением выбранного визуального направления.

## Добавлено

- Папки внутри раздела «Сферы жизни».
- Возвращены люди, заметки и книги как личные папки.
- Добавлена папка «Путешествия» с планированием бюджета.
- Возвращён календарь: события, задачи и поездки в одном расписании.
- Название приложения стабильно: Second Brain OS.
- Домен открывается без параметров версии.
- Старые версии не прогружаются слоями.
