# Second Brain OS V91 · FOCUSED HOTFIX

Дата сборки: 23.07.2026  
Build: `second-brain-space-v91-focused-hotfix-20260723-r1`

## Что исправлено

### 1. Читаемость карточки цели

Причина: глобальная тема синего hero-блока принудительно наследовала белый цвет всем вложенным подписям. Из-за этого текст внутри белого круга прогресса и белых плашек становился практически невидимым.

Исправление:
- текст прогресса получил тёмный контрастный цвет;
- подпись прогресса имеет контраст 4.95:1;
- метаданные цели получили тёмную полупрозрачную подложку;
- плашка «КАМПАНИЯ» получила читаемый тёмно-синий текст.

### 2. Очередь «Разобрать»

Причина: дополнительный обработчик повторно добавлял любую операцию со словом «перевод» в очередь даже после назначения конкретной категории. Кроме того, флаг `needsCategoryReview` не сбрасывался.

Исправление:
- после сохранения категории устанавливается `needsCategoryReview = false`;
- сохраняется `categoryReviewedAt`;
- операция добавляется в очередь только при реально отсутствующей или общей категории;
- карточка и счётчик обновляются сразу без перезагрузки.

### 3. Кнопка «Сохранить» в форме дохода

Причина: форма молча прерывала сохранение, если три процента распределения не давали ровно 100%. На пользовательском примере 70% + 10% + 10% получалось 90%, поэтому кнопка выглядела нерабочей.

Исправление:
- добавлен живой счётчик распределения;
- при сумме меньше 100% показано понятное объяснение;
- недостающий процент безопасно добавляется к «свободным деньгам»;
- при сумме больше 100% показывается ошибка внутри формы и фокус переводится к проблемному полю;
- данные сохраняются, окно закрывается, значения остаются после повторного открытия.
