# Second Brain OS V78 — Premium Unified

## Что изменено

- Единый светлый премиальный дизайн на всех разделах.
- Старый V77-интерфейс больше не загружается, поэтому слои версий не наслаиваются.
- Новый главный экран в стиле утверждённого макета.
- Информативные карточки папок: видно состав и показатели до открытия.
- Общая страница привычек и отдельная страница каждой привычки.
- Быстрые отметки одной или всех привычек.
- Возвращены подсказчик, интервью с подсознанием и вишлист привычек.
- Финансы: отдельные страницы «Операции», «Анализ категорий», «Экспорт CSV» и «Источники долгов».
- В операции выводится полный список с пагинацией по 25 записей.
- CSV содержит абсолютно все операции, а не только видимую страницу.
- Сумма долгов объясняется конкретными активными записями.
- Информация объединяет Людей, Заметки, Желания, Книги и остальные необязательные разделы без удаления данных.
- Создан персональный профиль: имя, фамилия, дата рождения, город, контакты, описание и фотография.

## Сохранность данных

V78 не очищает и не заменяет существующие коллекции. Миграция только добавляет:

- `state.settings.profile`
- `state.settings.v78`
- `state.habitWishlist`, если массива ещё не было
- базовую привычку чтения, только если привычки чтения не существовало

Перед установкой рекомендуется скачать JSON через кнопку «Экспорт данных».

## Установка

1. Загрузить изменённые файлы в корень репозитория.
2. Заменить `index.html` и `sw.js`.
3. Добавить `app-v78-premium.js` и `styles-v78-premium.css`.
4. Выполнить Commit changes.
5. Полностью закрыть вкладку приложения.
6. Открыть сайт и нажать `Ctrl + Shift + R`.

## Откат

Вернуть файлы из резервной копии V77. Личные данные при откате не удаляются, так как V78 использует совместимую структуру.
