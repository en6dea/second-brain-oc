# Second Brain OS V85 — GameLife Automation & HP

V85 развивает существующую V83.3 без очистки пользовательских данных и сохраняет утверждённый светлый премиальный стиль.

## Что добавлено

### Награды и HP
- создание, редактирование и удаление наград;
- стоимость награды в HP;
- эмодзи или URL изображения;
- категория и описание;
- минимальный уровень;
- лимит получений;
- период восстановления между получениями;
- включение и отключение награды;
- журнал начисления и списания HP;
- архивная копия перед удалением награды.

### Ежедневная автоматизация
- утренний check-in: состояние, главный фокус, напоминание;
- автоматический вечерний итог по привычкам, задачам, расходам и быстрым записям;
- недельный обзор с привычками, задачами, расходами и лучшим днём;
- мягкие автоматические предложения при открытии приложения в подходящее время.

### Быстрый ввод
Глобальная функция «Зафиксировать за 30 секунд» сохраняет мысль, эмоцию, задачу, идею, расход или событие с минимальным количеством полей.

### Очередь «Разобрать»
Автоматически собираются:
- операции без понятной категории;
- входящие записи;
- задачи без даты;
- цели без следующего действия;
- желания без ссылки или картинки;
- книги без общего количества страниц;
- приближающиеся дни рождения.

Система предлагает разбирать первые 3 элемента, чтобы не перегружать пользователя.

### Цели
- поиск целей без следующего действия;
- создание следующего шага из незавершённого этапа;
- создание задачи из шага цели;
- обнаружение целей без движения;
- сохранение прежних кампаний и этапов.

### Книги
- чтение связано с GameLife;
- отдельная сессия чтения с минутами, страницами и мыслью;
- автоматическое обновление прогресса;
- XP и HP за сессию;
- существующие цитаты, конспекты, оценки и прогресс сохранены.

### Достижения
Добавлены достижения за привычки, чтение, операции, цитаты, этапы целей, возвращение после пропуска и регулярные недельные обзоры.

## Сохранность данных
Миграция V85 только дополняет структуру состояния. Существующие записи не очищаются и не заменяются. Перед установкой всё равно рекомендуется скачать JSON-экспорт из приложения.
