# Second Brain OS V82

## Установка без старого кэша

1. В работающем приложении скачайте JSON через «Экспорт данных».
2. Загрузите все файлы из архива обновления в корень GitHub-репозитория с заменой.
3. Сделайте Commit changes и дождитесь завершения GitHub Pages.
4. Один раз откройте:
   `https://en6dea.github.io/second-brain-oc/force-update.html`
5. Страница удалит только старый кэш интерфейса и автоматически откроет V82. Личные записи не удаляются.

## Почему обновление теперь должно примениться

- Вся логика и стили находятся внутри одного `index.html`.
- Старые JS/CSS-файлы не подключаются и не могут реагировать на кнопки.
- `force-update.html` удаляет предыдущие service worker и кэши.
- Новый service worker использует отдельный cache name V82 и network-first для страниц.

## Основные изменения

- Симметричные карточки привычек.
- Выбор из 24 иконок при создании или редактировании привычки.
- Новый календарь: день, неделя, месяц и архив.
- Цвета событий по важности.
- Добавление события в Google Календарь.
- Новый последовательный подсказчик с очередью следующих шагов.
- Новый фирменный значок приложения.
- Исправления переноса и смещения текста в карточках.

## Сохранность данных

V82 не очищает `tasks`, `operations`, `habits`, `people`, `notes` и другие коллекции. Перед первым запуском создаётся автоматическая резервная копия состояния V81.1 в IndexedDB.
