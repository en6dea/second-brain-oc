# Second Brain OS V86.5 — связь целей и привычек

## Что изменено

Цели теперь поддерживают 3 независимые механики достижения:

1. **Только действия**
   - для проектов, покупок, регистраций, разовых результатов;
   - привычки не требуются;
   - прогресс считается по метрике и этапам.

2. **Через привычки**
   - для чтения, сна, спорта, финансового учёта и других повторяемых действий;
   - нужно связать минимум одну привычку;
   - основной показатель цели — стабильность связанных привычек.

3. **Гибрид**
   - цель движется этапами и задачами;
   - привычки поддерживают регулярность;
   - итоговый показатель: 70% прогресс кампании и 30% стабильность привычек.

## Где настраивается связь

- в мастере создания или редактирования цели;
- на странице цели через кнопку **«Связать привычки»**;
- в форме редактирования самой привычки через блок **«Связанные цели»**.

Одна привычка может поддерживать несколько целей. Одна цель может использовать несколько привычек. При этом привычка может оставаться самостоятельной, а цель — существовать без привычек.

## Сохранность данных

- существующим целям автоматически назначается режим **«Только действия»**;
- старые цели, задачи, привычки и отметки не очищаются;
- при удалении привычки связь с целями снимается автоматически;
- при выполнении связанной привычки цель получает дату последней активности;
- данные продолжают сохраняться в текущую локальную структуру приложения.
