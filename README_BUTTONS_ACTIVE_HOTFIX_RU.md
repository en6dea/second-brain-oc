# Second Brain OS — Buttons Active Hotfix

Точечная сборка для GitHub.

## Что исправлено

- Кнопки в долгах: `Напомнить`, `Ред.`, `Закрыть` теперь обрабатываются жёстким роутером действий.
- Кнопки категорий доходов/расходов: `Ред.` и `Удалить` активны.
- Удаление категории теперь не возвращает её обратно через операции: связанные операции переводятся в `Без категории`.
- Исправлен роутинг `data-action`, `data-id`, `data-scope`, `data-row`, `data-day`, `data-type`.
- Сохранён текущий дизайн Timeline Focus UI2.

## Файлы

Заменить в корне GitHub:

- `index.html`
- `app.js`
- `sw.js`
- `manifest.webmanifest`
- `offline.html`
- `force-update.html`

После загрузки открыть обычный домен или `force-update.html` для сброса кэша.
